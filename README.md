## 💡 Motivation

> “A small action can save a life that cannot speak for itself.”

FurEver Care was built with the belief that technology should not only solve human problems,  
but also extend care to voiceless animals in need.

Every report submitted here can potentially save a life.

---

## 🏗️ System Workflow

1. User uploads animal image 📸  
2. Adds location + description 📍  
3. Data is processed in Streamlit app ⚡  
4. (Future) AI evaluates urgency using Gemini 🧠  
5. Rescue alert is generated 🚨
