# 🐾 FurEver Care

### 💖 *Because every fur needs FurEver Care*

---

## 🚑 About the Project

**FurEver Care** is a simple yet impactful web application built using **Streamlit** that helps users report injured or stray animals quickly so that help can reach them faster.

The goal is to bridge the gap between people and animal rescue services using technology.

---

## ✨ Features

- 📸 Upload image of injured/stray animal  
- 📍 Enter location or landmark  
- 📝 Add incident description  
- 🚨 Submit rescue report  
- 🧠 (Future) AI-based injury severity detection using Gemini  
- 🗺️ (Future) Map integration for precise location tracking  

---

## 🛠️ Tech Stack

- Python 🐍  
- Streamlit ⚡  
- Google Gemini API 🤖  
- HTML/CSS (basic UI styling)  
- dotenv (for secure API key handling)
